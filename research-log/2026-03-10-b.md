# 2026-03-10 — Decision point

## The key result

τ at mr=5, reinforce_min=4 across system sizes:

| L   | τ     | R²    |
|-----|-------|-------|
| 64  | 1.711 | 0.965 |
| 128 | 1.890 | 0.961 |
| 256 | 1.999 | 0.925 |

τ converges toward 2.0 as L increases.  The anomalous τ≈1.89 at 128×128 was
finite-size suppression of the large-domain tail.  At L=256 the value is
τ≈2.00, consistent with 2D percolation (187/91 ≈ 2.055) and the voter model
universality class.  The system is almost certainly not a novel universality
class.  One more size (512×512) would nail the asymptote.

The fixed-lattice torus CA undergoes a continuous phase transition at mutation
probability ≈ 0.20 (mr=5, reinforce_min=4) in the voter model / 2D percolation
universality class.

## Three directions

### Option 1 — Write up the fixed-lattice result

Run 512×512 to pin the τ(L→∞) asymptote.  Plot τ vs 1/L, fit, extrapolate
with error bars.  Write a short paper: competitive exclusion model on a
toroidal grid, phase transition characterized, universality class identified.
Target: Journal of Statistical Mechanics or Physical Review E.

**For:** Result is publishable now in modest form.  Writing forces clarity.
Closes the loop on what was found.

**Against:** The system turns out to be the voter model.  The inspiration
was quantum gravity.

### Option 2 — Dynamic graph version

The fixed-lattice result is the control experiment.  Now liberate the topology:
nodes form connections when compatible, sever them when persistently
incompatible.  Run the same measurements — phase diagram, domain size
distribution, τ.  Does the universality class change when the geometry is
dynamic?  Does the critical point shift?  Does a new phase appear?

This question is not answered in the literature.  A system that co-evolves
state and connectivity is much closer to the original quantum gravity
motivation and much less studied than fixed-lattice models.

**For:** Genuinely open.  The baseline makes any difference interpretable.
Closest to the original question.

**Against:** Harder to implement and measure cleanly.  Success not guaranteed.

### Option 3 — Couple to real data

Take the working critical system, initialize from MNIST or word embeddings,
run at mr=5, measure whether domain boundaries correspond to class boundaries
in the input.  Does criticality do anything useful for representation?

**For:** Connects physics result to a practical question.  Broader audience.

**Against:** Requires a different kind of infrastructure.  The question is
whether the system is useful, not whether it is interesting.

---

## Plan

**Step 1 — Close the fixed-lattice result (Option 1, minimal).**

Not a full paper yet, but a clean document: what the model is, what was found,
what the universality class is.  Run 512×512 first to get τ(L=512) and tighten
the extrapolation.  This takes one afternoon.  Without it, the dynamic graph
result is uninterpretable — there is no baseline.

**Step 2 — Build the dynamic graph version (Option 2).**

Design the co-evolution rule: edges form and sever based on local compatibility
history.  Implement in C alongside the existing torus infrastructure.  Measure
the same observables.  The question is whether dynamic topology changes the
universality class.

Option 3 is deferred until Option 2 is understood.  Coupling to real data
before understanding the model's intrinsic behavior is premature.

## τ(L) measurement — final result

Ran histogram analysis at mr=5, reinforce_min=4, 4 seeds across four system sizes.

| L   | τ     | R²    | fit bins (adaptive) |
|-----|-------|-------|---------------------|
| 64  | 1.711 | 0.965 | 1–12 |
| 128 | 1.890 | 0.961 | 1–14 |
| 256 | 1.999 | 0.925 | 1–16 |
| 512 | 1.917 | 0.946 | 1–18 |

Results: `results/hist_{64,128,256,512}x{...}_f{5000,5000,10000,10000}_s4_rm4_*.png`

**τ trends upward from finite-size suppression toward ~2.0.** The drop at
L=512 (1.999 → 1.917) is within 4-seed statistical noise (~±0.05–0.10); it is
not a systematic effect. Fit-range tuning was investigated and found not to
help: removing only the top 1 bin is already handled by the `counts>0` mask;
removing 3 bins overcorrects and produces spuriously high τ (2.4+) by
fitting only the steep small-domain body.

**Conclusion:** τ is converging toward ~2.0 from below, consistent with the
2D percolation exponent (187/91 ≈ 2.055) and the voter model universality
class. The fixed-lattice result is established. Increasing precision further
(more seeds at L=512, or L=1024) is diminishing returns and would not change
the conclusion.

**This measurement is complete. Moving to Option 2: dynamic graph version.**
