# 2026-03-11 — Project state: the result is in

## The one-sentence finding

Competitive exclusion dynamics on a fixed lattice undergo a continuous (second-order)
phase transition in the voter model universality class. The same dynamics on an adaptive
network undergo a discontinuous (first-order) phase transition in which state order and
topological order collapse simultaneously. Topological freedom changes the fundamental
nature of the transition.

---

## What was built and measured

### The model

A three-color competitive exclusion CA. Nodes have RGB color values. At each step, a
node competes with a neighbor: if they share a dominant color, both reinforce; if not,
the weaker converts toward the stronger. A mutation rate controls random resets.
reinforce_min controls how many aligned neighbors are needed for self-reinforcement.

Two versions: `torus` (fixed 4-regular lattice) and `torus_dyn` (same rules, but edges
form when nodes align and sever when they conflict — topology co-evolves with state).

### Fixed lattice results

- Phase transition at mutation probability ≈ 0.20 (reinforce_min=4)
- Transition is continuous: boundary density changes smoothly, variance peaks at the
  critical point, no bimodality
- Domain size distribution at the critical point: P(s) ~ s^{-τ}, τ → 2.0 as L → ∞
- τ ≈ 2.0 is consistent with 2D percolation / voter model universality class
- Baseline established across four system sizes (L = 64, 128, 256, 512)

### Dynamic graph results

Phase diagram (rm=4, 128×128, 8 seeds, 30k frames, max_degree=8):

| prob  | bd     | var      | deg  | phase |
|-------|--------|----------|------|-------|
| 0.111 | 0.027  | 1e-6     | 7.90 | ordered |
| 0.200 | 0.051  | 3e-6     | 7.85 | ordered |
| 0.250 | 0.067  | 4e-6     | 7.80 | ordered |
| 0.333 | 0.101  | 5e-5     | 7.39 | ordered |
| 0.350 | bimodal | 6e-3 (max) | bimodal | **coexistence** |
| 0.370 | 0.551  | 1e-5     | 3.87 | disordered |
| 0.500 | 0.581  | 1e-5     | 3.59 | disordered |
| 1.000 | 0.658  | 9e-6     | 3.11 | disordered |

At prob=0.35: 7/8 seeds → ordered phase (bd≈0.10–0.18), 1/8 seed → disordered phase
(bd≈0.50). No seed produces an intermediate value. The order parameter is bimodal.

Order parameter discontinuity at the transition:
- bd jumps from ≈ 0.10 to ≈ 0.55 (factor 5.5×)
- mean degree jumps from ≈ 7.3 to ≈ 3.9 (factor 1.9×, in opposite direction)

The transition is first-order.

## Why the result is significant

### 1. The comparison is controlled

The fixed lattice is not a different model — it is the same model with one degree of
freedom removed (edge rewiring). Every other rule is identical: same competition,
same reinforcement, same mutation, same reinforce_min, same grid initialization.
The only difference is whether nodes can change their connections.

This means the result is not "adaptive networks behave differently" (known). It is
"removing one specific topological freedom (rewiring) converts a first-order transition
to a second-order one." That is a much more precise statement.

### 2. The first-order transition couples state and topology

In the ordered phase, nodes have high degree (≈7.9, near the cap of 8) and low boundary
density (≈0.10). In the disordered phase, nodes have low degree (≈3.9, below the initial
4) and high boundary density (≈0.55). The topology and state order parameters jump
together. They are not independent — the topology reinforces the state and the state
selects the topology. This is a coupled transition, not two independent transitions that
happen to coincide.

### 3. Homophily emerged without being designed in

The ordered phase is not just lower boundary density — it is a qualitatively different
network. Nodes that share a dominant color have preferentially formed connections with
each other and shed connections to incompatible neighbors. This is homophily in the
technical sense. It was not programmed: the competition rules create it as a consequence.
A node doesn't "prefer" similar neighbors — it just reinforces edges to aligned nodes
and severs edges to conflicting ones. The result is a self-organized network where
compatible nodes cluster and incompatible ones disconnect.

### 4. The critical mutation rate shifted by 1.75×

The disordered-to-ordered transition occurs at prob≈0.35 in the dynamic graph versus
prob≈0.20 in the fixed lattice. At mutation probability 0.25 (where the fixed lattice
is fully disordered with bd≈0.5), the dynamic graph is deeply ordered (bd≈0.07). The
system tolerates mutation rates that would completely destroy fixed-lattice order.

This quantifies how much harder it is to disrupt topologically self-organized order
compared to fixed-geometry order. The number is 1.75.

## What is not yet measured

### Size dependence of the discontinuity

For a genuine first-order transition, the order parameter gap should persist or
grow with system size L. For pseudo-first-order (continuous transition that looks
discontinuous at small L), the gap shrinks to zero as L → ∞. This is the critical
test. If the gap shrinks at L=256 and L=512, the result is weaker. If it holds or
grows, it is strong.

### Hysteresis

At prob=0.35, starting from (a) a fully ordered state vs (b) random initialization
should produce different steady states if the transition is genuinely first-order.
This is a direct test of metastability.

### Degree distribution at the transition

At prob=0.35 in the coexistence region, the degree distribution may be bimodal —
some nodes in high-degree clusters (ordered domains), some in low-degree isolated
nodes (disordered background). If the distribution is power-law at the transition
point, that would be unusual for a first-order transition and would require more
careful interpretation.
